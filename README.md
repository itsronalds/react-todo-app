## Technologies

* [React.js](https://react.dev/)
* [TypeScript](https://www.typescriptlang.org/)
* [Yarn](https://yarnpkg.com/)

## Clone

```$
git clone https://github.com/itsronalds/react-todo-app.git
```

## Install dependencies

```$
cd react-todo-app &&
yarn
```

## Start project
```$
yarn start
```
